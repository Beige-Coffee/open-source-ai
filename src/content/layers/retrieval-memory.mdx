The infrastructure that lets a model recall, search, and remember beyond
its <G term="context-window">context window</G>. Three things live here: vector databases (Qdrant,
Weaviate, Chroma, Milvus, LanceDB, pgvector), <G term="embedding">embedding</G> models
(sentence-transformers, BGE, jina-embeddings), and agent-memory
frameworks (Letta, mem0, Zep). The open side is robust at every
sub-layer; Pinecone is the closed counterpart and pgvector is the
boring-tech path for most teams. The interesting story at this layer is
not the database itself (vectors plus metadata is a portable format) but
the accumulated-context lock-in one layer up: an agent whose memory
lives server-side at someone else's cloud is not portable, no matter
how open the underlying vector format is.
