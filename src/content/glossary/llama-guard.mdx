A small open model trained to classify text against a taxonomy of
unsafe categories (violence, hate, sexual content, weapons, criminal
planning, etc). The intended deployment pattern is filtering both
user prompts and model responses: a <G term="llama">Llama</G> Guard call wraps each
generation, blocking or flagging outputs that violate the configured
policy.

The taxonomy is editable, and the model handles instruction-style
policy definitions reasonably well, so a team can configure
"acceptable" per their own use case rather than accepting Meta's
defaults. <G term="llama">Llama</G> Guard 3 (July 2024) expanded language coverage to
eight languages; the separately released Llama Guard 3 Vision
(November 2024) added image-input classification.

Full coverage at [/projects/llama-guard](/projects/llama-guard).
