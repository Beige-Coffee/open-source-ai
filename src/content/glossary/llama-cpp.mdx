The reference <G term="local-first">local-first</G> LLM runtime. <G term="llama">llama</G>.cpp started as a weekend
port of the original <G term="llama">LLaMA</G> model to C++ with hand-tuned ARM and AVX
kernels; by 2024 it covered virtually every modern <G term="open-weights">open-weight</G> model,
nearly every <G term="quantization">quantization</G> scheme via the `GGUF` format, <G term="gpu">GPU</G> offload
via Metal, <G term="cuda">CUDA</G>, <G term="rocm">ROCm</G>, and Vulkan, and an HTTP server compatible with
the OpenAI API.

<G term="ollama">Ollama</G>, <G term="lm-studio">LM Studio</G>, GPT4All, Jan, and most other consumer-facing local
runtimes are llama.cpp under different UIs. The project is permissively
licensed and community-maintained.

Full coverage at [/projects/llama-cpp](/projects/llama-cpp).
